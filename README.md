# Project-Alpaca---Mean-Reversion-Strategy
Systematic trend-filtered mean reversion on 5 cyclical stocks (MOS, LEN, LYB, DVN, HAL) via Alpaca paper trading — shared backtest/live engine, hard risk limits, cost modeling, Streamlit dashboard. Turned −7.6%/half-year into +1.8%/year net, validated out-of-sample. FINM 25000 project, co-authored by Jason Zou and Ignacio Lopez Amartino.
